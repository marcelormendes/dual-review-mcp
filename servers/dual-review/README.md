dual-review-mcp

Usage via NPX

Add to your repo:

```
{
  "mcpServers": {
    "dual-review": {
      "command": "npx",
      "args": ["-y", "dual-review-mcp@latest"]
    }
  }
}
```

In Cursor chat:
- Zero-arg: Run tool dual-review:dual-review
- Flexible: Run tool dual-review:run_dual_review {"secondaryCommand":"cursor"}
